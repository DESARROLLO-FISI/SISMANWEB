Funciones del Sistema:

En el panel izquierdo podemos encontrar 3 secciones : Curso, Alumno y Docente. Describiremos cada funcion.

Curso -> Registrar : Nos permite registrar un curso nuevo.

Curso -> Asignar Equivalencia : Nos permite asignar un curso a un conjunto de cursos coincidentes o crear un conjunto nuevo.

Curso -> Buscador : Nos permite buscar algun curso de acuerdo a distintos filtros.

Alumno -> Registro Individual : Nos permite registrar un alumno nuevo.

Alumno -> Registro Grupal : Nos permite registrar un grupo de alumnos a partir de un archivo.

Alumno -> Buscador : Nos permite buscar algun alumno de acuerdo a distintos filtros.

Docente -> Registro Individual : Nos permite registrar un docente nuevo.

Docente -> Registro Grupal : Nos permite registrar un grupo de docentes a partir de un archivo.

Docente -> Buscador : Nos permite buscar algun docente de acuerdo a distintos filtros.
